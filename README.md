# Tax Formalization Gap Audit (GitHub Pages)

This repository publishes the Markdown findings report as a polished, public
government-style site.

## Live doc

Once deployed, the main page renders:

- `index.html` (styled, searchable navigation, responsive layout)
- `outputs/tax-formalization-exploration.md` (source findings)

## Deploy

The repository includes a GitHub Pages workflow:

- `.github/workflows/gh-pages.yml`

The workflow builds from the main branch and publishes `index.html` and all
checked-in assets.
